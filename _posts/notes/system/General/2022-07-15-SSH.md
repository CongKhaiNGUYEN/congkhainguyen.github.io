# SSH

## Basic Access
We can try login using the **anonymous** user by default (with no password):
```console
$ ssh anonymous@ip
    Enter password:
anonymous@ip#  
```

## RSA file
If we manage to get the rsa file we can connect with:
```console
$ chmod 600 id_rsa
$ ssh -i id_rsa user@ip
```
